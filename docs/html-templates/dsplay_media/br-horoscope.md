# Daily Horoscope (Brazil)

```js
var dsplay_media = {
  // All general fields are also included...
  
  // Brazilian Daily Horoscope
  // (the `iteration` variable can be used to rotate between signs)
  title: 'Daily Horoscope',
  signs: [
    {
      name: "Aries",
      text: "bla bla bla ...",
    },
    {
      name: "Touro",
      text: "bla bla bla ...",
    },
    // ... same for the others
  ],
}
```
